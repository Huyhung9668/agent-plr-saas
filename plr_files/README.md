# PLR Files

Neu khong biet file PLR nen bo vao folder nao, cu vut tat ca vao:

```txt
plr_files/_INBOX_DROP_HERE/
```

Sau do chay menu:

```txt
15. Auto-sort PLR inbox
```

Agent se tu phan loai theo ten file:

- AI
- Marketing
- Planner
- Kids-Story
- Etsy
- Self-Help
- Health
- Business
- Unsorted

Neu ten file khong co dau hieu license nhu `PLR`, `MRR`, `rights`, `license`, `resell`, file se duoc dua vao:

```txt
plr_files/_Needs-License-Check/
```

Rule: file chua ro license chi nen dung de research idea, chua nen ban nguyen ban.
