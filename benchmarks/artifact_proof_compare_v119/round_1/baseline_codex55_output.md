# 01_Workflow_Map.md

The workflow has seven steps.  
1. Pick a narrow buyer: beginner freelancer, busy parent, teacher, coach, creator, or small shop owner.  
2. Pick one practical outcome: plan, track, organize, review, brainstorm, or prepare.  
3. Generate the printable structure with AI.  
4. Edit for clarity, originality, tone, and usefulness.  
5. Format into a simple PDF, Canva design, Google Doc, or KDP interior.  
6. Run the quality checklist before selling or giving away.  
7. Publish a small listing and test buyer interest.

Buyer test: show the mockup and title to 10 target buyers or run a small paid listing. If nobody clicks, saves, or asks questions, improve the angle before creating more pages.

# 03_Template_Guide.md

Use a simple product template so the buyer immediately understands the value. A strong 10-page printable pack can include: cover page, quick-start page, goal page, checklist page, tracker page, planning page, reflection page, resource page, weekly review page, and next-step page.

Design rules: use readable fonts, generous spacing, clear labels, and no clutter. Every page should help the buyer complete one action. Avoid decorative pages that add bulk but no utility.

For KDP interiors, keep margins safe, avoid tiny text, and test print readability. For PLR packs, include editable source files only if your license and creation process allow it. Always include a usage guide and compliance note.

# 05_Quality_Checklist.md

Before publishing, review every printable with this checklist.  
- Is the buyer specific enough to recognize themselves?  
- Does each page produce a useful decision, list, plan, or record?  
- Are instructions short and clear?  
- Are there spelling, grammar, or formatting issues?  
- Is the design readable in black and white?  
- Does the file export correctly as PDF?  
- Is the size appropriate for the intended use?  
- Are claims safe and realistic?  
- Did you avoid medical, legal, financial, or guaranteed income advice?  
- Did you create enough original value beyond raw AI text?

AI replace risk: medium. Reduce it by adding examples, page usage notes, buyer-specific workflows, and better formatting.
