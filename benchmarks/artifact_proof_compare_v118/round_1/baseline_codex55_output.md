# 01_Workflow_Map.md

The workflow has seven steps. First, choose one safe niche and buyer. Example: “busy teachers who need simple rainy-day coloring pages.” Second, generate 20 page concepts with consistent style rules. Third, create image prompts with clear black-line printable instructions. Fourth, review outputs for quality, originality, age fit, and print usability. Fifth, organize the best pages into a simple pack with cover, title, and usage notes.

Sixth, create listing copy using the included sales kit. Seventh, run the launch audit before offering the product anywhere. Do not upload unreviewed AI outputs directly. Check for distorted hands, messy lines, unreadable details, accidental logos, or unsafe content. The goal is a practical asset pack with clean use cases, not a random folder of AI images.

# 05_Quality_Checklist.md

Before using or selling any printable page, run this checklist.

1. The page is black-and-white and printable without gray muddy areas.  
2. No text appears unless intentionally added and proofread.  
3. No logos, watermarks, signatures, or accidental brand marks are visible.  
4. The art does not resemble a famous character, brand, celebrity, or franchise.  
5. The line art is clean enough for coloring.  
6. The page is age-appropriate for the stated buyer.  
7. The file has correct dimensions for the intended use.  
8. The license terms of every AI tool, font, graphic, and template are reviewed.  
9. The product description does not promise income, ranking, therapy, diagnosis, or guaranteed outcomes.  
10. A refund policy, support contact, and delivery instructions are included.

If any item fails, fix before listing.

# 06_Fix_Prompts.md

Use these prompts when your AI output is messy.

**Fix Prompt 1: Cleaner Lines**  
Revise the image prompt to create cleaner black outline coloring art. Remove gray shading, heavy shadows, clutter, text, logos, watermarks, and tiny details. Keep the subject centered and printable.

**Fix Prompt 2: Safer Originality**  
Rewrite this coloring page idea so it avoids trademarks, famous characters, copyrighted worlds, celebrity likenesses, and brand references. Keep the general theme but make it original and generic.

**Fix Prompt 3: Better Buyer Fit**  
Improve this printable pack for a specific buyer. Buyer: elementary teacher needing simple classroom coloring pages. Make the concepts easy to print, non-controversial, and suitable for classroom use.

**Fix Prompt 4: Listing Compliance**  
Review this sales copy for risky claims. Remove income guarantees, therapy claims, fake scarcity, fake testimonials, and unsupported statements. Keep it persuasive but honest.
