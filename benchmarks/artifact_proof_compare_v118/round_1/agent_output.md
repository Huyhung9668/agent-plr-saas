# 00_Start_Here.md

Chào mừng bạn đến với Printable Prompt-to-Product Sprint. Hãy bắt đầu bằng cách chọn một loại sản phẩm: coloring page pack, kids worksheet, journal interior, planner sheet, coach lead magnet, hoặc Canva printable bundle. Không chọn quá rộng. # 01_Workflow_Map.md

Quy trình 7 bước: 1) chọn micro-niche; 2) xác định buyer và use case; 3) tạo prompt sản xuất nội dung; 4) tạo layout/template guide; 5) test 3 output mẫu; 6) chạy checklist lỗi; 7) đóng gói thành ZIP có README, prompt, ví dụ, listing copy và license note. Ví dụ: buyer là coach muốn lead magnet, output là “5-page printable habit tracker”. Input gồm audience, goal, style, page count, size. Output cuối gồm PDF sample, editable guide, prompt pack, checklist và listing description. Không launch khi chưa kiểm tra trademark, font, stock, Canva và quyền AI-image.
