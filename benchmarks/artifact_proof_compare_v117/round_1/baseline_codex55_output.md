# Prompt Library

## 1. Product Angle Prompt
Act as a digital product strategist. Create 10 narrow printable product angles for this buyer: [BUYER].  
Constraints: simple to make, non-regulated, no income claims, useful within 7 days.  
Output: product name, buyer pain, promise, 10-page pack idea, price suggestion.

## 2. Page Set Builder Prompt
Create a 15-page printable pack for: [PRODUCT ANGLE].  
Include: page title, purpose, fields/sections, user instructions, design notes.  
Avoid copyrighted brands, medical/legal/financial advice, and unrealistic promises.

## 3. KDP Interior Prompt
Create a low-content KDP interior concept for: [NICHE].  
Output: title idea, subtitle, 30 interior page types, repeatable page structure, buyer benefit, compliance risks.

## 4. PLR Rebrand Prompt
Transform this generic printable idea into a sharper PLR product: [IDEA].  
Output: new product name, target buyer, improved promise, 5 bonuses, license caution, sales angle.

## 5. Listing Copy Prompt
Write marketplace listing copy for: [PRODUCT].  
Include: headline, subheadline, benefits, what’s included, who it’s for, FAQ, refund note.  
Do not claim guaranteed results.

## 6. Quality Review Prompt
Review this printable pack: [DESCRIPTION].  
Score 1–10 for usefulness, clarity, buyer fit, design simplicity, compliance risk, and refund risk.  
Give fixes before launch.

## 7. Bonus Builder Prompt
Create 5 practical bonuses for a printable product about: [TOPIC].  
Bonuses must be checklists, trackers, prompt sheets, examples, or templates.

## 8. WarriorPlus Funnel Prompt
Create a simple funnel for this printable/PLR product: [PRODUCT].  
Include front-end, order bump, OTO1, OTO2, backend, affiliate angle, compliance notes.
