# Workflow Map

## Step 1: Choose Buyer

Use this formula:

> I help [buyer] create/use [printable outcome] without [main frustration].

Examples:

- Homeschool parents → weekly lesson planners
- New Etsy sellers → printable habit trackers
- Busy freelancers → client project trackers
- Beginner KDP sellers → niche journal interiors

## Step 2: Pick Product Type

Choose one:

- Planner
- Tracker
- Journal
- Checklist
- Workbook
- Activity pack
- Swipe/template pack

## Step 3: Generate Structure

Use AI to create:

- Product title ideas
- Page list
- Page instructions
- Design notes
- Buyer use case
- Listing description

## Step 4: Build the Printable

Recommended tools:

- Canva
- Google Docs
- PowerPoint
- Affinity Publisher
- Adobe Express

## Step 5: Quality Check

Check:

- Page size
- Margins
- Spelling
- Duplicate pages
- Commercial-use image rights
- No unrealistic claims
- No copyrighted/trademarked brands

## Step 6: Package

Include:

- PDF printable
- Editable Canva template, if permitted
- Usage guide
- License notes
- Support FAQ

## Step 7: Test Offer

Before full launch, run a small buyer test:

- Send to 5–10 target buyers
- Ask if they understand the outcome
- Ask what they would pay
- Ask what page/template is missing
- Collect objections

# Prompt Library

## Prompt 1: Niche Finder

Act as a digital product strategist.  
Give me 20 printable product ideas for this buyer: [BUYER].  
Constraints: simple to create, non-regulated, no income claims, suitable for Etsy/KDP/PLR.  
Output table: product idea, buyer pain, printable pages, difficulty, risk.

## Prompt 2: Product Angle

Create 10 fresh angles for a printable product about [TOPIC].  
Avoid generic planner names.  
Make each angle specific to one buyer and one outcome.

## Prompt 3: Page List Builder

Create a page-by-page outline for a [PRODUCT TYPE] for [BUYER] who wants [OUTCOME].  
Include page title, purpose, fields needed, and design notes.

## Prompt 4: Canva Design Brief

Write a Canva design brief for this printable: [PRODUCT].  
Include page size, style, fonts mood, color palette, spacing, icon style, and accessibility notes.

## Prompt 5: KDP Interior Plan

Create a KDP low-content interior plan for [NICHE].  
Include trim size, page count suggestion, repeating page types, front matter, and compliance risks.

## Prompt 6: PLR Pack Builder

Turn this printable concept into a PLR-style product pack.  
Include main product, usage guide, license notes, sales description, bonuses, and buyer FAQ.

## Prompt 7: Listing Description

Write a marketplace listing for [PRODUCT].  
Avoid income claims.  
Include buyer problem, what’s included, how to use it, file format, and disclaimer.

## Prompt 8: Quality Reviewer

Review this printable product concept for legal, quality, buyer clarity, and refund risk.  
Return: pass/fail, problems, fixes, launch readiness score.
