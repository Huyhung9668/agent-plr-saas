## FILE: 03_Template_Guide.md

```md
# Template Guide

## Standard worksheet page layout
- Top: worksheet title
- Under title: one-sentence instruction
- Middle: activity area
- Bottom: optional parent/teacher note
- Footer: page number or brand name

## Page types
1. Trace: dotted letters/numbers
2. Match: draw line between related items
3. Count: count objects and write number
4. Circle: find correct answer
5. Color: simple coloring activity with instruction

## Canva build notes
Use A4 or US Letter. Keep 0.25–0.5 inch margins. Use readable fonts. Avoid tiny details.
```
