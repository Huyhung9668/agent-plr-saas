# Workflow Map

## Step 1: Pick One Buyer

Use this formula:

> I help [specific buyer] create/use [printable type] to solve [small pain].

Examples:

- Busy moms → weekly meal planning sheets  
- Beginner freelancers → client onboarding checklist  
- Teachers → classroom behavior tracker  
- Etsy sellers → product idea planner  
- Authors → character profile workbook

## Step 2: Choose One Printable Format

Pick one:

- Planner
- Tracker
- Checklist
- Workbook
- Journal
- Prompt cards
- KDP interior
- PLR worksheet pack

## Step 3: Build the Pack

Minimum viable pack:

- Cover page
- How to use page
- 5 printable pages
- 1 checklist
- 1 example completed page
- License/compliance page

## Step 4: Quality Control

Check:

- No medical/legal/financial claims
- No copied branded names
- No fake results
- No trademarked character references
- No “guaranteed income” claims
- Print margins are readable
- Page titles are clear

## Step 5: Package

Export as:

- PDF for buyers
- Editable Canva/PowerPoint/Google Docs version if allowed
- PNG preview images
- License text file
- Delivery instructions

# Prompt Library

## Prompt 1: Micro-Niche Finder

Act as a digital product strategist.  
Input: [broad niche].  
Generate 10 micro-niche printable pack ideas.  
For each, include buyer, pain, printable type, promise, and risk level.  
Avoid medical, legal, financial, trademark, or guaranteed income claims.  
Output as a table.

## Prompt 2: Printable Pack Builder

Act as a printable product creator.  
Buyer: [buyer]  
Pain: [pain]  
Printable type: [planner/tracker/checklist/workbook]  
Create a 7-page printable pack outline.  
Each page must have title, purpose, fields, and user instructions.

## Prompt 3: Page Copy Generator

Create copy for this printable page:  
Page title: [title]  
Buyer: [buyer]  
Goal: [goal]  
Fields needed: [fields]  
Tone: clear, simple, practical  
Output: heading, short instruction, field labels, example entry.

## Prompt 4: KDP Interior Variation

Turn this printable pack into a KDP-style interior.  
Create 30 page variations using repeated but useful layouts.  
Avoid copyrighted references, exaggerated claims, or professional advice.

## Prompt 5: PLR Rebrand Angle

Create 5 rebrand angles for this printable pack.  
Include target buyer, new title, promise, bonus idea, and listing hook.  
Do not claim income or sales.

## Prompt 6: Listing Copy

Write a marketplace listing for this printable pack.  
Include title, subtitle, description, what’s included, who it’s for, how to use, disclaimer, and refund note.  
Avoid fake scarcity and guaranteed results.

# Fix Prompts

## Fix Generic Output

Rewrite this printable page to be more specific for [buyer].  
Remove vague advice.  
Add practical fields, short instructions, and one example entry.

## Fix Risky Claims

Review this copy for risky claims.  
Remove guaranteed income, health, legal, financial, or professional advice language.  
Replace with safer educational wording.

## Fix Weak Listing

Improve this product listing.  
Make the buyer clearer, explain what’s included, add use cases, and remove hype.

## Fix Low-Value Printable

This printable feels too thin.  
Add fields, checkboxes, examples, and a short “how to use” note.

## Fix AI-Sounding Copy

Rewrite this in plain human language.  
Use short sentences.  
Avoid hype, clichés, and vague motivational phrases.
