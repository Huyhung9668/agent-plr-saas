# Prompt Library

## Prompt 1 - Niche Selector
Role: You are a printable product strategist.
Input: buyer=[...], platform=[Etsy/KDP/Lead Magnet], skill level=[beginner].
Task: suggest 10 worksheet product ideas.
Constraints: avoid medical, legal, income, therapy, trademarked themes.
Output: table with niche, buyer pain, worksheet type, risk, differentiation.

## Prompt 2 - 5-Page Worksheet Builder
Create a 5-page printable worksheet pack for:
buyer=[...]
topic=[...]
age/group=[...]
tone=[calm/practical/friendly]
Output:
1. Cover page text
2. Instructions page
3. Worksheet 1
4. Worksheet 2
5. Reflection/checklist page
Include simple instructions and no professional claims.

## Prompt 3 - Canva Layout Guide
Turn this worksheet content into a Canva layout plan.
Output columns: page, section, font style, spacing, icons/illustrations, export notes.
Avoid copyrighted characters, brands, or trademarked design elements.

## Prompt 4 - Listing Copy
Write Etsy/KDP/Gumroad listing copy for this worksheet pack.
Include: title, short description, 5 bullets, what’s included, who it is for, disclaimer.
Avoid income guarantees, educational guarantees, or fake scarcity.

## Prompt 5 - Fix Prompt
Review this worksheet for: unclear instructions, risky claims, duplicate content, weak buyer benefit, missing disclaimer, AI-generic language.
Return: issue, severity, rewrite suggestion.
