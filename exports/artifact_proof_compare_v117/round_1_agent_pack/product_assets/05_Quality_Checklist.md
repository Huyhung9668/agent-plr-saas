# Quality Checklist

Before selling or sharing:
- [ ] Buyer is specific, not “everyone”.
- [ ] Topic avoids medical/legal/financial/therapy claims.
- [ ] No copyrighted characters, celebrities, brands, or lyrics.
- [ ] Instructions are clear without needing extra coaching.
- [ ] Each page has one job.
- [ ] Font is readable when printed.
- [ ] AI text has been edited for natural language.
- [ ] Includes “not professional advice” note where relevant.
- [ ] Canva/font/stock licenses checked.
- [ ] Listing does not promise sales, ranking, or guaranteed outcomes.
