```md
# Template Guide

Recommended printable structure:
- Page 1: Cover + outcome
- Page 2: Quick instructions
- Page 3: Main exercise
- Page 4: Practice/reflection
- Page 5: Review checklist

Design rules:
- Use large margins for print.
- Keep black-and-white version if targeting KDP.
- Avoid tiny fonts under 10pt.
- Use original icons or licensed Canva elements only.
- Export PDF print + PNG preview.

Template placeholders:
[Product Name]
[Buyer Type]
[Worksheet Goal]
[Instruction Text]
[Exercise Box]
[Reflection Prompt]
[Disclaimer]
```
