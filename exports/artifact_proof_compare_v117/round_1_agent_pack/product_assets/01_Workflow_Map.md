```md
# 7-Step Worksheet Product Workflow

| Step | Action | Output |
|---|---|---|
| 1 | Pick buyer + use case | Buyer statement |
| 2 | Choose safe niche | Non-regulated topic |
| 3 | Generate worksheet structure | 5-page outline |
| 4 | Create page content | Instructions + exercises |
| 5 | Format in Canva/Docs | Printable draft |
| 6 | QC + compliance check | Fix list |
| 7 | Create listing/sales copy | Title, description, bullets |

Example niche:
Buyer: beginner productivity coach  
Use case: client lead magnet  
Product: Weekly Focus Worksheet Pack  
Safe promise: help users plan weekly priorities more clearly, not guarantee productivity results.
```
