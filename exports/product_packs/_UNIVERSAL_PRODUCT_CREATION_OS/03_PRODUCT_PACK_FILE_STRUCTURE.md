# Universal Product Pack File Structure

Use this folder structure for most PLR, AI, SaaS, template, prompt, swipe, checklist, or training products.

```text
Product-Name/
├── 00-README.txt
├── 01-Start-Here.pdf
├── 02-Core-System.pdf
├── 03-Use-Case-Examples.pdf
├── 04-Templates-or-Swipes.docx
├── 05-Customization-Prompts.pdf
├── 06-Implementation-Checklist.pdf
├── 07-Planner-Sheet.xlsx
├── 08-Bonus-Asset.pdf
├── 09-License-And-Compliance-Notes.pdf
└── 10-Upgrade-Path-And-Next-Steps.pdf
```
