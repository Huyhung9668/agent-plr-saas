# Topic Upgrade Requirements

Use these requirements for every future topic.

## Email Marketing Topic

Must include:

- Welcome use case
- Nurture use case
- Promo use case
- Affiliate use case
- Deadline/urgency use case
- Re-engagement use case
- Subject lines
- CTA bank
- Customization prompt
- Pre-send checklist
- Campaign planner
- Compliance note

## PLR Rebrand Topic

Must include:

- PLR evaluation checklist
- License/risk check
- Repositioning worksheet
- Product name/angle generator
- Offer stack builder
- Bonus generator
- Sales page outline
- Delivery ZIP structure
- WarriorPlus launch checklist

## SaaS/Tool Topic

Must include:

- Tool promise
- User workflow
- Input/output examples
- Use-case demos
- Onboarding checklist
- Pricing model
- Feature scope
- Support/risk note
- Upgrade roadmap

## Affiliate/JV Topic

Must include:

- Affiliate avatar
- Promo angle
- Email swipe pack
- Bonus stack
- JV page outline
- Commission note
- Launch calendar
- Approval criteria
- Disclosure/compliance note

## Sales Page Topic

Must include:

- Hook
- Pain
- Consequence
- Mechanism
- Offer stack
- Benefit bullets
- Proof substitute
- Objection handling
- FAQ
- Guarantee/refund language
- CTA
- Compliance check
