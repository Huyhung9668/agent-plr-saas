# Beginner Email Campaign Checklist

Use this checklist before sending each email or campaign.

## 1. Audience Check

| Question | Yes/No |
|---|---|
| Do I know who this email is for? | |
| Is the reader a beginner, intermediate, or advanced user? | |
| Did I mention a problem they actually care about? | |
| Is the email written in simple language? | |

## 2. Message Check

| Question | Yes/No |
|---|---|
| Does the email focus on one main idea? | |
| Is the subject line clear? | |
| Is the opening sentence easy to understand? | |
| Is the email short enough to read quickly? | |
| Did I remove unnecessary hype? | |

## 3. Offer Check

| Question | Yes/No |
|---|---|
| Is the offer or next step clear? | |
| Did I explain who the product is for? | |
| Did I avoid promising guaranteed results? | |
| Did I include the correct link? | |
| If this is an affiliate email, did I include disclosure? | |

## 4. CTA Check

| Question | Yes/No |
|---|---|
| Is there only one main CTA? | |
| Is the CTA easy to follow? | |
| Does the CTA match the email goal? | |
| Did I avoid pressure if there is no real deadline? | |

## 5. Compliance Check

| Risky Item | Avoided? |
|---|---|
| Guaranteed sales | |
| Copy-paste profits | |
| Instant commissions | |
| Fake deadline | |
| Fake scarcity | |
| Misleading income claim | |
| Missing affiliate disclosure | |

## 6. Final Send Check

Before sending, confirm:

- The subject line matches the email.
- The placeholders are replaced.
- The link works.
- The email has one main purpose.
- The tone is helpful.
- The offer is realistic.
- The deadline is real if mentioned.
