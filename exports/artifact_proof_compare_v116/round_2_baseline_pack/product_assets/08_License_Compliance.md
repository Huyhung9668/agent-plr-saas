# Workflow Map

## Step 1: Choose Buyer

Use this formula:

> I help [buyer] create/use [printable outcome] without [main frustration].

Examples:

- Homeschool parents → weekly lesson planners
- New Etsy sellers → printable habit trackers
- Busy freelancers → client project trackers
- Beginner KDP sellers → niche journal interiors

## Step 2: Pick Product Type

Choose one:

- Planner
- Tracker
- Journal
- Checklist
- Workbook
- Activity pack
- Swipe/template pack

## Step 3: Generate Structure

Use AI to create:

- Product title ideas
- Page list
- Page instructions
- Design notes
- Buyer use case
- Listing description

## Step 4: Build the Printable

Recommended tools:

- Canva
- Google Docs
- PowerPoint
- Affinity Publisher
- Adobe Express

## Step 5: Quality Check

Check:

- Page size
- Margins
- Spelling
- Duplicate pages
- Commercial-use image rights
- No unrealistic claims
- No copyrighted/trademarked bran
