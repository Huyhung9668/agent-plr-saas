# Template Guide

## Recommended printable pack structure
- Cover page
- How to use page
- 10–30 core printable pages
- Bonus tracker/checklist
- License/disclaimer page

## Canva/Design rules
- Use commercially safe fonts only.
- Avoid Disney, Marvel, celebrities, sports teams, brand names.
- Keep margins print-friendly.
- Export PDF Print for buyers, PNG preview for listing.

## KDP/Etsy adaptation
KDP: interior consistency, trim size, no misleading claims.  
Etsy: preview images, clear buyer instructions, digital download note.  
WarriorPlus: sell the system, not guaranteed marketplace results.
