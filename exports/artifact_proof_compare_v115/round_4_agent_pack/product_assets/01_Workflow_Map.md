# 5-Step Printable Lead Magnet Sprint

| Step | Action | Output |
|---|---|---|
| 1 | Pick buyer | Buyer + pain statement |
| 2 | Choose lead magnet type | Checklist / worksheet / planner |
| 3 | Generate draft | AI-assisted content draft |
| 4 | Format into printable | Clean sections + placeholders |
| 5 | QA + publish | Opt-in copy + delivery page draft |

Recommended lead magnets:
- Coach: Client Readiness Checklist
- Freelancer: Project Intake Worksheet
- Etsy seller: Product Listing Planner
- Affiliate: Bonus Claim Checklist
- PLR reseller: Rebrand Planner
