# Prompt Library

## Prompt 01 - Lead Magnet Concept Finder
Role: You are a digital product strategist.
Input: niche, buyer, current offer, audience level.
Task: Generate 10 printable lead magnet ideas.
Constraints: practical, non-hype, no income guarantees.
Output format: table with title, buyer, pain, promise, format, CTA.
Quality check: flag ideas that are too broad or generic.

## Prompt 02 - Checklist Builder
Create a 1-page checklist for [buyer] who wants [outcome].
Include:
- title
- 7-12 checkable items
- short instructions
- common mistakes
- CTA line
Avoid vague advice. Each item must be verifiable.

## Prompt 03 - Worksheet Builder
Create a printable worksheet for [topic].
Sections:
1. Goal
2. Current problem
3. Input fields
4. Decision table
5. Action plan
6. Review score
Use placeholders and one filled example.

## Prompt 04 - Mini Planner Builder
Build a 5-page mini planner for [buyer] to complete [outcome] in [timeframe].
Pages: cover, goal map, daily plan, tracker, review.

## Prompt 05 - Rewrite PLR Into Printable
Transform this source outline into an original printable worksheet.
Do not copy sentences. Extract workflow only.
Add examples, checklist, and compliance note.
Source outline: [paste outline]

## Prompt 06 - Quality Fix Prompt
Audit this printable for: unclear promise, vague steps, missing examples, hype, legal/health/finance claims, weak CTA.
Return: issue, severity, rewrite suggestion.
