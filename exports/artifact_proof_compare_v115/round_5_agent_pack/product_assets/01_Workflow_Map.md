```md
# 7-Step Printable Lead Magnet Workflow

1. Buyer Match
- Ai dùng?
- Họ muốn hoàn thành việc gì?
- Họ đang bị kẹt ở đâu?

2. Lead Magnet Promise
Format: Help [buyer] achieve [small outcome] without [obstacle].

3. Asset Type
- Checklist: nhanh, dễ dùng
- Worksheet: tương tác, giá trị cao hơn
- Mini planner: hợp productivity/marketing
- Cheat sheet: hợp prompt/tool/process

4. AI Draft
Dùng prompt theo format. Không dùng nguyên bản.

5. Human Edit
Xóa claim quá đà, thêm ví dụ, thêm bước cụ thể.

6. Design Prep
Tạo layout 3-7 trang: cover, how to use, core pages, next step.

7. Deploy
Dùng làm opt-in, bonus affiliate, order bump, hoặc mini product.
```
