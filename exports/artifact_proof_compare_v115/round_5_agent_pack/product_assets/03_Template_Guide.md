```md
# Template Guide

Recommended printable structures:

## 1-Page Checklist
- Header: outcome promise
- Use instruction: “Complete before…”
- 10 checkbox items
- Mistakes box
- CTA footer

## 3-Page Worksheet
Page 1: Problem + goal
Page 2: guided fields + decision prompts
Page 3: action plan + review score

## 5-Page Mini Planner
Cover
Outcome Map
Step Planner
Progress Tracker
Review + Next Step

Design Rules:
- Use large margins
- Keep one task per page
- Avoid tiny text
- Add “For educational use” note if needed
- Do not include copyrighted logos, brands, characters, quotes, or protected images
```
