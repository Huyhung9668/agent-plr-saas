# SaaS Files

Thu muc nay dung de luu tat ca tai lieu lien quan den huong micro SaaS.

Neu khong muon tu phan loai, chi can vut tat ca file vao:

```txt
saas_files/_INBOX_DROP_HERE/
```

Sau do chay menu:

```txt
14. Auto-sort SaaS inbox
```

## Folder

```txt
saas_files/
  _INBOX_DROP_HERE/  vut tat ca file SaaS vao day
  ideas/          SaaS ideas tu PLR analysis
  specs/          product specs, MVP specs, feature docs
  mockups/        UI sketches, screenshots, wireframes
  code_samples/   code mau, snippets, prototype notes
  research/       competitor research, market notes
  pricing/        pricing tiers, LTD, agency offer
  validation/     beta feedback, launch notes, buyer questions
  whitelabel/     agency, client mode, whitelabel notes
```

## Rule

PLR raw files de trong `plr_files/`.
SaaS planning/build files de trong `saas_files/`.
