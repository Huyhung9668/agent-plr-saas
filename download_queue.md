# PLR Download Queue

Use this file to track forum threads or vendor pages you want to research.

Important:
- Only download files you are allowed to access and use.
- Do not sell unchanged files when the license is unclear.
- Save the thread URL, title, visible license note, and reason it looks useful.
- After downloading, keep the original license/readme file beside the product.
