# Product Launch Agent System

This system splits the product launch workflow into three focused agents.
Each agent has its own input folders, step outputs, and final deliverable.

## BUILD PRODUCT

- Role: Product builder agent for PLR, MRR, RR, prompt packs, templates, mini courses, and digital assets.
- Mission: Turn raw source material into a packaged, deliverable digital product with bonuses, backend offers, and an improvement path.
- Input root: `input_files/agent_workflows/01_build_product`
- Final output: A complete product pack: core product, bonuses, delivery structure, upsell/backend plan, and version roadmap.

Steps:

01. Nghien cuu san pham -> `01_product_research`
02. Len y tuong va dinh vi -> `02_idea_positioning`
03. Tao san pham so -> `03_digital_product_creation`
04. Xay dung noi dung -> `04_content_building`
05. Dong goi san pham -> `05_product_packaging`
06. Tao bonus -> `06_bonus_creation`
07. Thiet ke va trinh bay -> `07_design_presentation`
08. Setup giao hang -> `08_delivery_setup`
09. Tao upsell backend -> `09_upsell_backend`
10. Cai tien san pham -> `10_product_improvement`

## JV MANAGER

- Role: Launch partner manager for affiliate recruitment, JV coordination, traffic pushes, and long-term relationships.
- Mission: Prepare and manage the affiliate/JV side of a launch from recruitment through post-launch follow-up.
- Input root: `input_files/agent_workflows/02_jv_manager`
- Final output: A launch partner system: prospect list, outreach, calendar, promo assets, support scripts, contest, and relationship tracker.

Steps:

01. Tim affiliate doi tac -> `01_affiliate_partner_research`
02. Outreach va ket noi -> `02_outreach_relationship`
03. Lap ke hoach launch -> `03_launch_planning`
04. Chuan bi promo materials -> `04_promo_materials`
05. Ho tro affiliate -> `05_affiliate_support`
06. Leaderboard va contest -> `06_leaderboard_contest`
07. Quan ly traffic va promo -> `07_traffic_promo_management`
08. Sau launch -> `08_post_launch`
09. Quan ly network JV lau dai -> `09_long_term_jv_network`

## SALE PAGE

- Role: Direct-response sales page agent for offer research, copy, page structure, funnel pages, and conversion testing.
- Mission: Turn offer notes and product assets into conversion-focused sales pages and funnel copy with realistic claims.
- Input root: `input_files/agent_workflows/03_sale_page`
- Final output: A complete front-end sales page plus funnel page copy, CTA map, FAQ, guarantee, and optimization checklist.

Steps:

01. Nghien cuu offer va buyer -> `01_offer_buyer_research`
02. Xay offer -> `02_offer_building`
03. Headline va hook -> `03_headline_hook`
04. Viet sales copy -> `04_sales_copy`
05. Trinh bay san pham -> `05_product_presentation`
06. Bonus stack -> `06_bonus_stack`
07. Guarantee va FAQ -> `07_guarantee_faq`
08. CTA va conversion elements -> `08_cta_conversion_elements`
09. Funnel pages -> `09_funnel_pages`
10. Toi uu va test -> `10_optimization_testing`
