# WarriorPlus Funnel Playbook

Use when planning FE, order bump, OTO, downsell, backend, WarriorPlus listing, pricing, and launch readiness.

Default funnel:
- FE: implementation kit, usually $17-$27.
- Order bump: speed asset, swipe bank, CTA bank, prompt bank, or checklist pack.
- OTO1: more depth, niche-specific DFY examples, advanced templates.
- OTO2: agency/client use, whitelabel, or commercial license if rights are clear.
- Recurring: monthly club, template drop, campaign club, or tool access.

Rule:
- FE sells workflow and implementation.
- Bump sells speed.
- OTO1 sells depth or more examples.
- OTO2 sells license or scale.
- Recurring sells ongoing supply/support.

Launch readiness:
- Product depth, sales page, JV materials, delivery, funnel, traffic, risk/compliance.
- Under 7/10 means soft launch only.
- Above 8/10 can be public launch.
