# SaaS Upgrade Playbook

Use when turning a digital product into SaaS, tool, membership, or whitelabel offer.

SaaS analysis:
- What repeatable workflow can be automated?
- What inputs does the user provide?
- What outputs does the tool generate?
- Can outputs be exported as DOCX, PDF, CSV, ZIP, or client report?
- Is there recurring value?

MVP plan:
- Phase 1: internal tool that generates markdown/CSV.
- Phase 2: user-facing upload/input form and saved projects.
- Phase 3: paid SaaS with usage limits and history.
- Phase 4: agency/whitelabel client reports.

Pricing:
- Basic: $19/month.
- Pro: $49/month.
- Agency: $97/month or higher if client exports are valuable.

License caution:
- Whitelabel only if you own the content or PLR license explicitly allows it.
