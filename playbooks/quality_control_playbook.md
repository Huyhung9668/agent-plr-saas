# Quality Control Playbook

## Anti-Generic Filter
If output is only prompts, templates, subject lines, emails, or generic advice, it is not agent-grade.

## Required Upgrade
Add:
- Verdict.
- Scorecard.
- AI Replace Risk.
- Workflow.
- Checklist.
- Planner/sheet.
- Example/case study.
- Sales page angle.
- Funnel.
- JV pack.
- SaaS upgrade.
- Agent status.
- Next actions.

## Critic Rule
Score below 8/10 if missing funnel, JV, SaaS, file output, or anti-raw-AI explanation.
