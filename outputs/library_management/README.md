# Library Management

This folder is the control layer for the local PLR/SaaS/WarriorPlus library.

## Files
- `Product_Index.csv`: quick inventory of product ideas/assets and risk.
- `Product_Build_Queue.md`: launch priority list.
- `license_vault/`: provenance and license review notes.

## Rule
Library products are inputs, not final outputs. Any PLR/MRR/RR source must be checked for license/provenance and rebuilt into original workflow, checklist, planner, examples, sales assets, and launch assets before selling.
